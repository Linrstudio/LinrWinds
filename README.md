LinrWinds是一款IE内核下类似于Firebug的页面调试插件。它适用于所有版本的IE（IE6、IE7、IE8、IE9、IE10）、绿色版的IE（如IEtester、IE Tab）和其他以IE为内核（Trident）的浏览器（如360、傲游、腾讯TT、世界之窗）-----一次安装，全部可以使用。安装后，在浏览器右键菜单中，选择“点亮网页”即可启动它。或点击右键后，快速按下D键，立即开启。
